@par Example Description

OPT SW Control is a Python application that must be used together with the 
firmware example running on a supported STM32 Nucleo board to configure and 
profile Opus encoder and decoder.


@par Directory contents

  - AudioTestSamples: 		set of audio test samples that can be used to 
		      		evaluate the profiling tool.
  - OPT_QuickStartGuide.pdf:    set of slide that explains how to use the tool.
  - OPT_Setup.exe:		OPT_SW_Control installer.
